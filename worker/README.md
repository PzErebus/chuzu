# 脚手架租赁管理系统

## 项目描述

这是一个完整的脚手架租赁管理系统，用于管理脚手架等物资的租赁业务。系统包含前端和后端两部分，实现了从物资管理、客户管理到开单、收款的完整业务流程。

## 系统架构

### 前端
- **技术栈**：Vue 3 + Vant UI + ECharts + HTML2Canvas + XLSX
- **功能模块**：
  - 首页：显示日租金预计、在外物资总数、资产分布图表、最近业务
  - 开单：支持送货出库和收货入库
  - 物资：物资列表和分布查询
  - 客户：客户管理和财务信息
  - 管理后台：物资管理、客户管理、系统配置

### 后端
- **技术栈**：Cloudflare Workers风格的API
- **数据库**：SQLite (D1)
- **主要接口**：
  - 系统配置管理
  - 管理员验证
  - 物资CRUD操作
  - 客户管理
  - 订单处理
  - 付款登记
  - 统计数据
  - 物资分布查询
  - 客户财务信息

## 功能特点

1. **完整的租赁业务流程**：从物资管理、客户管理到开单、收款
2. **数据可视化**：使用ECharts展示资产分布、订单趋势、客户分析等
3. **单据生成**：支持生成和保存单据图片用于打印
4. **Excel导出**：支持导出客户财务报表
5. **响应式设计**：适配移动端设备，支持触摸操作
6. **数据安全**：管理员密码验证和操作日志记录
7. **自动数据库维护**：确保数据库结构完整
8. **物资分类管理**：支持对物资进行分类管理，方便查询和统计
9. **租金计算和账单生成**：自动计算租金并生成详细账单
10. **库存预警**：当物资库存低于阈值时发出预警
11. **数据导入导出**：支持JSON格式的数据导入导出，方便数据迁移和备份
12. **物资损坏/维修记录**：记录物资的损坏和维修情况，跟踪维修状态
13. **通知系统**：系统操作和重要事件的通知提醒
14. **操作日志**：详细的系统操作记录，用于安全审计
15. **系统备份和恢复**：支持完整的系统数据备份和恢复
16. **客户分类和信用等级**：对客户进行分类管理和信用评级
17. **批量操作**：支持批量修改和删除物资、客户等数据
18. **数据统计和报表**：多维度的数据统计和分析报表

## 快速开始

### 本地开发

1. 克隆仓库
```bash
git clone <仓库地址>
cd chuzu
```

2. 部署后端API
   - 后端API设计为Cloudflare Workers风格，可部署到Cloudflare Workers
   - 也可以使用其他支持类似API结构的平台

3. 配置前端
   - 前端代码已内置API调用逻辑，默认使用相对路径

4. 部署前端
   - 前端可以部署到GitHub Pages、Vercel、Netlify等静态网站托管服务

### 系统初始化

1. 首次访问系统时，数据库会自动创建所需的表结构
2. 默认管理员密码为 `admin`
3. 建议登录后修改管理员密码

## 部署指南

## 使用说明

1. **登录管理后台**：点击首页右上角的设置图标，输入管理员密码（默认：admin）
2. **添加物资**：在管理后台的物资管理中添加物资信息，包括名称、规格、库存、租金、原值等
3. **添加客户**：在管理后台的客户管理中添加客户信息，包括姓名、电话、项目、地址等
4. **开单**：在开单页面选择客户和物资，提交生成单据
5. **查看客户财务**：在客户列表中点击客户，查看客户的财务信息和在租物资
6. **登记收款**：在客户财务页面登记收款记录
7. **查看统计数据**：在管理后台的数据统计页面查看业务概览、物资分析和客户分析

## 技术文档

### API接口

- **系统配置**
  - **GET /config**：获取系统配置
  - **POST /config**：保存系统配置

- **管理员验证**
  - **POST /auth**：管理员验证

- **物资分类管理**
  - **GET /categories**：获取分类列表
  - **POST /categories**：添加分类
  - **PUT /categories**：更新分类
  - **DELETE /categories**：删除分类

- **客户分类管理**
  - **GET /customer-categories**：获取客户分类列表
  - **POST /customer-categories**：添加客户分类
  - **PUT /customer-categories**：更新客户分类
  - **DELETE /customer-categories**：删除客户分类

- **物资管理**
  - **GET /products**：获取物资列表
  - **POST /products**：添加物资
  - **PUT /products**：更新物资
  - **DELETE /products**：删除物资
  - **GET /products/low-stock**：获取库存预警列表
  - **GET /products/:id/distribution**：获取物资分布

- **客户管理**
  - **GET /customers**：获取客户列表
  - **POST /customers**：添加客户
  - **PUT /customers**：更新客户
  - **DELETE /customers**：删除客户
  - **GET /customers/:id/financial**：获取客户财务信息

- **订单管理**
  - **POST /orders**：创建订单
  - **GET /orders/:id**：获取订单详情

- **付款管理**
  - **POST /payments**：登记付款

- **账单管理**
  - **GET /bills**：获取账单列表
  - **POST /bills**：生成账单
  - **PUT /bills**：更新账单状态

- **维修记录**
  - **GET /product-repairs**：获取维修记录
  - **POST /product-repairs**：添加维修记录
  - **PUT /product-repairs**：更新维修记录

- **数据导入导出**
  - **GET /export**：导出数据
  - **POST /import**：导入数据

- **数据统计**
  - **GET /stats**：获取统计数据
  - **GET /stats/detailed**：获取详细统计数据
